# CyberSecurity - Site de studiu NIST SP 800-61r2

Site interactiv pentru pregătirea examenului de **Cyber Security**, bazat pe documentul **NIST SP 800-61r2** (Computer Security Incident Handling Guide).

## Accesare online 

Site-ul este live pe GitHub Pages:

**https://ale0204.github.io/CyberSecurity/**


## Ce conține

- **Pagina principală** - 11 scenarii de incidente cu întrebări și răspunsuri
- **Scenarii detaliate** (1-11) - fiecare scenariu cu întrebări individuale, referințe NIST și răspunsuri ascunse
- **Exerciții practice** - 7 scenarii practice suplimentare
- **Simulare examen** - 5 scenarii tip role-play
- **Rezumate NIST** - Capitolele 2 și 3 din NIST SP 800-61r2

## Accesare locală (opțional)

Dacă vrei să rulezi site-ul local pe calculatorul tău:

```bash
# 1. Cloneaza repo-ul
git clone https://github.com/ale0204/CyberSecurity.git

# 2. Intra in folder
cd CyberSecurity

# 3. Porneste un server local cu Python
python -m http.server 8080

# 4. Deschide in browser
# http://localhost:8080
```

Sau simplu: deschide fișierul `index.html` direct în browser.
